Selenidoku
==========

Play Sudoku with the [Selenium](http://seleniumhq.org) automation tool!

How to play
-----------

    easy_install pip
    sudo pip install selenium
    python selenidoku.py

Notes
-----

This script hits http://www.websudoku.com/ continuously. Please use
respectfully.
